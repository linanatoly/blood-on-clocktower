# 血染钟楼 功能点汇总

> 按上帝/玩家双视角分列全部用户功能，用于与市面同类产品横向对比。

---

## 一、上帝视角

### 1. 创建与配置房间

| # | 功能 | 说明 |
|---|------|------|
| 1.1 | 创建房间 | 设置 4 位数字房间号，一键开房 |
| 1.2 | 设置总人数 | 滑块拖动选择 5-15 人 |
| 1.3 | 阵营配置 | 分别设置村民/外来者/爪牙/恶魔的人数，自动校验总和 |
| 1.4 | 阵营自动分配 | 调整总人数时自动按比例搭配阵营 |

### 2. 准备阶段

| # | 功能 | 说明 |
|---|------|------|
| 2.1 | 角色分配模式切换 | 随时在「玩家自选」和「上帝指定」之间切换，即刻生效 |
| 2.2 | 上帝为玩家指定角色 | 点玩家座位→选角色，逐一分配 |
| 2.3 | 设置酒鬼标记 | 点玩家座位标记/取消酒鬼，同时只能标记一人 |
| 2.4 | 查看玩家准备状态 | 已准备的玩家座位显示绿色「已准备」标记 |
| 2.5 | 全员准备后开局 | 所有玩家准备完毕即可点击开始游戏 |
| 2.6 | 解散房间 | 关闭房间全员返回大厅，需二次确认 |

### 3. 游戏中

| # | 功能 | 说明 |
|---|------|------|
| 3.1 | 推进昼夜阶段 | 点击「下一阶段」按序列前进（第一夜→第一日→…→第二十日） |
| 3.2 | 回退阶段 | 点击「上一阶段」可回退，方便纠正误操作 |
| 3.3 | 实时阶段显示 | 顶部状态栏中文显示当前阶段（如"第三日"） |
| 3.4 | 设置玩家存活状态 | 点玩家座位设为：存活/死亡有票/死亡无票 |
| 3.5 | 死亡状态视觉区分 | 死亡玩家座位显示 ☠️ 蒙层、红色划线，有票/无票样式不同 |
| 3.6 | 游戏中设酒鬼标记 | 游戏过程中同样可标记/取消酒鬼 |
| 3.7 | 结束游戏 | 手动结束游戏进入复盘阶段 |

### 4. 事件记录（专属）

| # | 功能 | 说明 |
|---|------|------|
| 4.1 | 快捷模板 | 17 个预设模板：死亡/被处决/攻击/下毒/保护/调查/开枪/提名/占卜/得票/被视为/变成 等 |
| 4.2 | 点选玩家填入 | 选模板后弹出玩家列表，点击即填 |
| 4.3 | 角色选择器 | 身份类事件弹出角色列表选择 |
| 4.4 | 数字微调器 | 得票类事件可调整数字（0-15） |
| 4.5 | 自定义文字记录 | 直接输入任意文字 |
| 4.6 | 彩色事件标签 | 自动标注类型（死亡/复活/阶段/开局/记录），不同颜色区分 |
| 4.7 | 修改与删除 | 已有记录可编辑或删除 |
| 4.8 | 仅上帝可见 | 全程对玩家隐藏 |
| 4.9 | 复盘可回看 | 游戏结束后仍可查看全部记录 |

### 5. 聊天

| # | 功能 | 说明 |
|---|------|------|
| 5.1 | 与任意玩家私聊 | 点玩家座位→聊天，一对一沟通（如夜间告知行动结果） |
| 5.2 | 发起群聊 | 勾选多名玩家同时对话 |
| 5.3 | 会话列表 | 所有聊天会话集中管理 |
| 5.4 | 未读角标 | 新消息时聊天按钮和会话均有未读提示 |

### 6. 复盘阶段

| # | 功能 | 说明 |
|---|------|------|
| 6.1 | 全员角色公开 | 所有玩家真实角色全员可见 |
| 6.2 | 回顾事件记录 | 可完整查看游戏全程事件 |
| 6.3 | 返回准备 | 一键回到准备阶段，清空角色和状态，开下一局 |

---

## 二、玩家视角

### 1. 加入房间与入座

| # | 功能 | 说明 |
|---|------|------|
| 1.1 | 加入房间 | 输入房间号和昵称即可加入 |
| 1.2 | 自由取昵称 | 自定义昵称显示 |
| 1.3 | 选择空座位入座 | 点击空座位确认入座 |
| 1.4 | 换座 | 已入座可选其他空位直接更换 |
| 1.5 | 显示座位编号 | 每个座位有数字编号（1-N） |
| 1.6 | 自己的座位高亮 | 自己的座位有发光效果，一眼可辨 |
| 1.7 | 退出房间 | 可随时离开房间返回大厅 |

### 2. 角色选择

| # | 功能 | 说明 |
|---|------|------|
| 2.1 | 阵营分类浏览 | 村民/外来者/爪牙/恶魔分四栏，每栏有标题 |
| 2.2 | 角色头像 | 每个角色有专属头像，直观选择 |
| 2.3 | 自行选择角色 | 在「玩家自选」模式下，点击座位自行挑选角色 |
| 2.4 | 等待上帝分配 | 在「上帝指定」模式下，看到"等待上帝分配角色"提示，直到上帝为自己选好 |
| 2.5 | 查看自己角色 | 选好后可在自己座位上看到角色名和头像 |

### 3. 准备与开始

| # | 功能 | 说明 |
|---|------|------|
| 3.1 | 准备/取消准备 | 入座并选好角色后，点击准备按钮 |
| 3.2 | 智能提示 | 按钮自动判断状态并切换文案——未入座→"请先入座"；未选角→"请选择角色"；等待上帝→"等待上帝分配角色"；就绪→"✅ 准备" |
| 3.3 | 误操作拦截 | 未入座或未选角点准备均有提示，不会出错 |

### 4. 游戏中

| # | 功能 | 说明 |
|---|------|------|
| 4.1 | 查看阶段 | 顶部状态栏显示当前昼夜阶段 |
| 4.2 | 查看自己的状态 | 自己存活/死亡状态在座位上可见 |
| 4.3 | 查看他人死亡状态 | 死亡玩家座位显示 ☠️ 蒙层和划线，有票/无票可区分 |
| 4.4 | 角色信息保密 | 只能看到自己的角色，看不到他人角色 |
| 4.5 | 对他人做角色推测 | 游戏中点其他玩家座位→选角色，做本地备注，仅自己可见 |

### 5. 聊天

| # | 功能 | 说明 |
|---|------|------|
| 5.1 | 与其他玩家私聊 | 点其他玩家座位→聊天，一对一沟通 |
| 5.2 | 与上帝私聊 | 点上帝座位即可私聊上帝 |
| 5.3 | 接收群聊 | 上帝发起的群聊自动出现在会话列表中 |
| 5.4 | 未读角标 | 新消息时按钮和会话列表均有提示 |
| 5.5 | 回车发送 | 输入消息后回车即可发送 |

### 6. 间谍模式（间谍角色专属）

| # | 功能 | 说明 |
|---|------|------|
| 6.1 | 一键开启间谍视角 | 游戏中显示「🕵 间谍模式」按钮 |
| 6.2 | 查看所有人的真实角色 | 开启后座位上直接显示所有玩家的角色 |
| 6.3 | 自动关闭 | 游戏结束或退出时自动关闭 |

### 7. 复盘阶段

| # | 功能 | 说明 |
|---|------|------|
| 7.1 | 查看所有角色 | 所有玩家角色公开显示 |
| 7.2 | 查看他人死亡/存活状态 | 复盘时完整查看游戏结果 |

## 三、全局体验（上帝和玩家共享）

| # | 功能 | 说明 |
|---|------|------|
| 1 | 断线自动重连 | 网络断开或关页面后重新打开，自动回到原房间 |
| 2 | 重连恢复全量数据 | 重连后恢复：座位、角色、聊天记录、事件记录（上帝） |
| 3 | 断线不掉队 | 断线期间保留座位和角色，标记断线但不踢出 |
| 4 | 连接状态显示 | 顶部状态栏提示「已连接」或「尝试重连中」 |
| 5 | 按钮智能显隐 | 根据准备/游戏中/复盘自动切换可见按钮 |
| 6 | 弹窗遮罩关闭 | 点弹窗外部空白区域即可关闭 |
| 7 | 移动端适配 | 适配手机屏幕，键盘弹出不遮挡内容 |
