## 1. 创建房间

1. 打开游戏页面，默认进入**大厅**。
2. 点击顶部 **「上帝」** 标签页。
3. 填写以下信息：
   - **房间号**：4 位数字，与玩家约定好，玩家凭此号加入。
   - **玩家人数**：拖动滑块设置（5-15 人）。
   - **阵营配置**：设置四个阵营的人数——村民、外来者、爪牙、恶魔。四个数字之和必须等于总人数。
4. 确认无误后，点击 **「创建房间」** 按钮。
5. 进入游戏页面，等待玩家加入。

> 💡 房间号建议用不容易被猜到的数字组合，避免陌生人误入。

---

## 2. 准备阶段

进入游戏页面后，底部操作栏会显示上帝专用的功能按钮。顶部状态栏显示当前阶段（"准备阶段"）和房间号。

### 2.1 等待玩家加入

- 玩家加入后，游戏桌面上会出现对应数量的座位卡牌。
- 玩家需要**入座**（点击座位→坐这）并**选择角色**（或等待上帝分配），最后点击**准备**。
- 每位玩家准备后，其座位上会显示 "已准备" 标记。
- 当所有玩家准备完毕，你就可以开始游戏了。

### 2.2 角色分配模式

在准备阶段，你可以切换角色分配方式。底部操作栏有 **「模式：玩家自选」** / **「模式：上帝指定」** 按钮：

| 模式 | 说明 |
|------|------|
| **玩家自选**（默认） | 玩家自己点击座位→「设置角色」选择角色 |
| **上帝指定** | 玩家无法自己选角色，由你（上帝）为每位玩家分配 |

点击按钮即可实时切换，切换后所有玩家的弹窗会自动关闭。

### 2.3 玩家自选模式

在此模式下，玩家自行选择角色。你的职责是监督和等待，不需要额外操作。你可以通过聊天功能与玩家沟通。

### 2.4 上帝指定模式

在此模式下，你需要为每位已入座的玩家分配角色：

1. **点击**某个已入座的玩家座位。
2. 在弹出的座位功能面板中，点击 **「设置角色」**。
3. 在角色选择界面中选择一个角色，点击**确认**。
4. 重复以上步骤，为每位玩家分配角色。
5. 玩家收到角色后，可以自行点击**准备**。

> ⚠️ 如果玩家还没有被分配角色就点击准备按钮，会看到「等待上帝分配角色」的提示，无法准备。

### 2.5 设置酒鬼标记

在准备阶段，你可以标记某位玩家为**酒鬼**（酒鬼不知道自己真实身份）：

1. 点击该玩家的座位。
2. 在弹窗中点击 **「他是酒鬼」**。
3. 该玩家头像右下角会出现 🍷 酒鬼图标。
4. 再次点击 **「取消酒鬼」** 可移除标记。

> 酒鬼标记互斥：同一时间只有一位玩家可以被标记为酒鬼。标记新玩家会自动取消旧的标记。

## 3. 开始游戏

所有玩家准备完毕后：

1. 点击底部 **「▶ 开始游戏」** 按钮。
2. 游戏正式开始，进入 **第一夜（night_1）** 阶段。
3. 底部按钮自动切换为游戏中的状态（阶段切换、结束游戏等）。

---

## 4. 游戏进行中

游戏开始后，你的操作栏会变化：

### 4.1 推进昼夜阶段

游戏按以下顺序推进：

```
第一夜 → 第一日 → 第二夜 → 第二日 → …… → 第十九夜 → 第二十日
```

底部有两个按钮：

| 按钮 | 功能 |
|------|------|
| **◀ 上一阶段** | 回退到上一个阶段（用于纠正误操作） |
| **▶ 下一阶段** | 推进到下一个阶段 |

顶部状态栏实时显示当前阶段名称（如"第一夜""第三日"）。

> 在夜间，你应该依次唤醒需要夜间行动的角色（如恶魔、占卜师等），通过私聊告诉他们行动结果。在日间，主持讨论、提名和投票。

### 4.2 设置玩家存活/死亡状态

点击玩家座位，弹窗中会出现三个状态按钮：

| 按钮 | 含义 |
|------|------|
| **存活** | 玩家正常存活 |
| **死亡有票** | 玩家死亡但仍有一票投票权（如被处决前的遗言票） |
| **死亡无票** | 玩家死亡且无投票权（常规死亡状态） |

游戏桌面上，死亡玩家会显示 ☠️ 死亡标记和划线。

### 4.3 事件记录器

点击底部 **「📋 记录」** 按钮打开事件记录器。这是一个上帝专属的日志工具，用于追踪游戏中的关键事件。

**记录事件的方式：**

- **快捷模板**：从预设列表中点击一个模板（如"死亡""被处决""下毒""调查"等），然后选择涉及的玩家即可快速生成一条记录。
- **自定义记录**：直接在输入框中输入任意文字记录。

模板分类：

| 类别 | 模板示例 |
|------|----------|
| 单玩家事件 | 死亡、被处决、没有死亡、无外来者 |
| 双玩家事件 | 攻击、下毒、保护、调查、开枪、提名、占卜、选择 |
| 数字事件 | 得票（{n}票）、获得数字 |
| 身份事件 | 被视为、变成、得知了 |

每条记录会自动标注类型标签（死亡/复活/阶段/开局/记录）并以不同颜色区分。

**管理事件：**
- 记录可随时**修改**或**删除**。
- 所有事件记录**仅你（上帝）可见**，玩家看不到。
- 复盘阶段事件记录会保留，方便回顾。

> 📝 事件记录器是你说书的重要辅助工具。建议在每次夜间行动后及时记录，避免遗漏关键信息。

### 4.4 聊天功能

点击底部 **「💬 聊天」** 按钮打开聊天面板。上帝支持两种聊天方式：

| 类型 | 说明 |
|------|------|
| **私聊** | 点击玩家座位→「聊天」，一对一与该玩家私聊。用于夜间告知行动结果 |
| **上帝群聊** | 在聊天面板中创建群聊，同时与多位玩家沟通 |

聊天面板顶部有未读消息提示。聊天记录会在你重连时自动同步恢复。

## 5. 结束游戏与复盘

当游戏分出胜负后：

1. 点击底部 **「🏁 结束游戏」** 按钮。
2. 游戏进入**复盘阶段**。此时：
   - 所有玩家的**真实角色公开**，所有人都能看到彼此的角色。
   - 事件记录可完整回看。
   - 无法再修改玩家状态或记录新事件。
3. 复盘结束后，你可以选择：
   - **「↩ 返回准备」**：回到准备阶段，清空所有角色和状态（保留角色分配模式设置），开始新一轮游戏。
   - **「解散房间」**：彻底关闭房间，所有玩家被移出。

---

## 6. 解散房间

在**准备阶段**，底部显示 **「解散房间」** 按钮：

1. 点击「解散房间」。
2. 弹出确认框：「确定要解散房间吗？所有玩家将被移出房间，房间数据将被清除」。
3. 点击「确认解散」。
4. 所有玩家返回大厅，房间数据被清除。

> ⚠️ 解散房间**不可撤销**，请确认所有玩家都已退出后再操作。

---

## 7. 断线重连

- 如果不小心关闭页面或网络断开，**不要慌**。
- 重新打开页面，游戏会自动尝试重连。
- 重连成功后，你会**回到原房间**，所有玩家、角色、事件记录和聊天记录完整恢复。
- 顶部状态栏显示连接状态：绿色"已连接"正常，黄色"尝试重连中"表示正在恢复连接。

### 准备阶段

| 按钮 | 作用 | 何时可见 |
|------|------|----------|
| 💬 聊天 | 打开聊天面板 | 始终 |
| 📋 记录 | 打开事件记录器 | 始终 |
| 模式：玩家自选/上帝指定 | 切换角色分配模式 | 准备阶段 |
| ▶ 开始游戏 | 开始游戏 | 所有玩家准备后 |
| 解散房间 | 解散当前房间 | 准备阶段 |

### 游戏中

| 按钮 | 作用 | 何时可见 |
|------|------|----------|
| ◀ 上一阶段 | 回到上一阶段 | 游戏中 |
| ▶ 下一阶段 | 推进到下一阶段 | 游戏中 |
| 🏁 结束游戏 | 结束游戏进入复盘 | 游戏中 |
| 💬 聊天 | 打开聊天面板 | 始终 |
| 📋 记录 | 打开事件记录器 | 始终 |

### 复盘阶段

| 按钮 | 作用 | 何时可见 |
|------|------|----------|
| ↩ 返回准备 | 清空角色回准备阶段 | 复盘阶段 |
| 💬 聊天 | 打开聊天面板 | 始终 |
| 📋 记录 | 查看事件记录 | 始终 |

### 座位弹窗（点击玩家座位）

| 按钮 | 作用 | 何时可见 |
|------|------|----------|
| 设置角色 | 为玩家选角色 | 准备阶段（上帝指定模式） / 游戏中（点非自己玩家） |
| 聊天 | 与该玩家私聊 | 座位有人且不是你 |
| 存活 | 设为存活状态 | 游戏中 |
| 死亡有票 | 设为死亡有票 | 游戏中 |
| 死亡无票 | 设为死亡无票 | 游戏中 |
| 他是酒鬼/取消酒鬼 | 切换酒鬼标记 | 准备阶段和游戏中 |
